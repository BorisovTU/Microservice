модуль HMACSHA.d32



int hmacsha(string message, snring key, string hash, int ver);

Вычисляет значение Хэша для переданной строки по алгоритмам   SHA1, SHA224, SHA256, SHA384, SHA512


Возвращаемое значение: код ошибки (0 - ошибки нет). В случае не нулевого кода ошибки также в параметр hash записывается ее описание.
Параметры: 
   message - строка, для которой будет вычисляться хэш
   key - строка с ключевым (секретным) словом
   hash - строка в которую возвращается значение хэш в шестнадцатиричном виде или описание ошибки
   ver - число с весрией алгоритма (допустимы значения 1,224,256,384,512), если параметр не задан, то используется алгоритм SHA256.
