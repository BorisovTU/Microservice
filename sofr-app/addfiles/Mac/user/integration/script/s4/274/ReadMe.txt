Процедура по заполнению таблиц 
RSHB_F303_2CHD_Generate.prc - сама процедура, входящие данные:
startdate        IN DATE   - начало периода, за который выбираются данные;
enddate          IN DATE   - конец периода, за который выбираются данные.
В выборку попадают сделки РЕПО, заключенные в выбранный период. 
При вызове процедуры таблицы очищаются и заполняются новыми данными.

6_РЕПО_ф303_Счета.sql - скрипт, создающий таблицу tRSHB_F303_2CHD_A;
5_РЕПО_ф303_Документы.sql - скрипт создающий таблицу tRSHB_F303_2CHD_P;
4_РЕПО_ф303_Сделки.sql - скрипт создающий таблицу tRSHB_F303_2CHD_D.
