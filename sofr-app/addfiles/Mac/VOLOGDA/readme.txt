Вологодский макрос загрузки списковых зачислений
================================================

Для того, чтобы данный макрос вызывался по комбинации клавиш Alt-M 
в АРМ "Универсальный фронт-офис" в списке платежных поручений по безналичным 
платежам, необходимо выполнить следующие действия:
- добавить каталог $(MAC)\VOLOGDA в список каталогов макрофайлов RS-Retail
  ПЕРЕД каталогом $(MAC)\DEPOSITR
- переименовать файлы 
  - $(MAC)\VOLOGDA\trn_lu_d_vologda.mac в $(MAC)\VOLOGDA\trn_lu_d.mac
  - $(MAC)\VOLOGDA\trn_lu_b_vologda.mac в $(MAC)\VOLOGDA\trn_lu_b.mac
- переместить файл $(MAC)\VOLOGDA\lgxs.d32 в $(OBJ)
- настроить в sbbank.ini путь к текстовым файлам списков 
  (переменная Retail4Users). Также возможна (но не обязательна) настройка 
  отдельных директорий для текстовых файлов, поступающих из различных 
  источников (переменные PFSobes, PFBezrab, PFWPens, PFZarp, PFStipTU, 
  PFZPB13, PFZPNew, PFDivSb)
